# Open Source Radar — 2026-07-07

## Main takeaway

В этом периоде доминируют высокозвёздные агентные платформы и AI-воркспейсы: NousResearch/hermes-agent (~211k звёзд), affaan-m/ECC (~227k) и pewdiepie-archdaemon/odysseus (~81k) выделяются как готовые к развёртыванию решения. Из сидов подтверждены Flowise, Activepieces, OpenMontage и другие классические open-source SaaS-альтернативы.

## Executive summary

Период 2026-07-07 насыщен зрелыми агентными продуктами с огромным тракшном. Лидеры по звёздам — affaan-m/ECC (227k, +1002) и NousResearch/hermes-agent (211k, +1768) — это agent harness и AI-агент-платформы, готовые к деплою и ребрендингу. pewdiepie-archdaemon/odysseus (81k, +708) — самый быстрорастущий self-hosted AI workspace, окончивший watch-фазу. Сиды подтверждают наличие готовых ниш: видео (OpenMontage), автоматизация (Flowise, Activepieces), аватары (Duix-Avatar), реклама (OpenAdServer), UGC (Open-AI-UGC). Основной риск — многие топ-репо являются фреймворками/библиотеками, а не готовыми продуктами; чистых деплоибельных продуктов в данных меньше, чем инструментов разработки.

- **Final score:** 8.4
- **Confidence:** high
- **Test steps:**
  1. git clone https://github.com/FlowiseAI/Flowise
  2. docker-compose up -d (или npx flowise start)
  3. Открыть http://localhost:3000, добавить API-ключи LLM
